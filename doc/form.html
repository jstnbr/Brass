Form

Brass provides a fundamental structure to form elements that can be customized using grid and utilities for styling.

Basic

The .form class creates a basic stacked structure for inputs. 

<form class="form">
  <label>Input</label>
  <input type="text">
  <label>Select</label>
  <select>
    <option>1</option>
    <option>2</option>
    <option>3</option>
  </select>
  <label>Textarea</label>
  <textarea rows="3"></textarea>
</form>

Color

Change the color of inputs by setting the border-color using CSS, or import a color library such as Colors. 

<form class="form">
  <label>Default</label>
  <input class="mb-1" type="text">

  <label>Focused</label>
  <input class="border--blue mb-1" type="text">

  <label>Success</label>
  <input class="border--green mb-1" type="text">

  <label>Required</label>
  <input class="border--red mb-1" type="text">

  <button class="btn bg-blue white" type="submit">Submit</button>
</form>

Inline

Set forms inline using the .form-i class. 

<form class="form form-i">
  <label>Search</label>
  <input type="text">
  <button type="submit" class="btn bg-gray">Submit</button>
</form>

Horizontal

Inputs can be organized into rows and columns to create horizontal style forms. 

<form class="form row">
  <div class="sm-c2">
    <label>Input</label>
  </div>
  <div class="sm-c10">
    <input class="mb-1" type="text">
  </div>
  <div class="sm-c2">
    <label>Select</label>
  </div>
  <div class="sm-c10">
    <select class="mb-1">
      <option>1</option>
      <option>2</option>
      <option>3</option>
    </select>
  </div>
  <div class="sm-c2">
    <label>Textarea</label>
  </div>
  <div class="sm-c10">
    <textarea class="mb-1"></textarea>
  </div>
  <div class="sm-c10 sm-offs-2">
    <label class="mb-1">
      <input class="v-mid" type="checkbox"> Checkbox
    </label>
  </div>
  <div class="sm-c10 sm-offs-2">
    <button class="btn bg-gray" type="submit">Submit</button>
  </div>
</form>

Sizing

Four different form input size options are available including the default, and three additional sizes .form-sm, .form-md, and .form-lg. 

<form class="form form-sm mb-1">
  <label>Input</label>
  <input type="text">
</form>

<form class="form mb-1">
  <label>Input</label>
  <input type="text">
</form>

<form class="form form-md mb-1">
  <label>Input</label>
  <input type="text">
</form>

<form class="form form-lg mb-1">
  <label>Input</label>
  <input type="text">
</form>
