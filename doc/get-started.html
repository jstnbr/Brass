Get Started

Use the default minified CSS file, or customize the scale and breakpoint settings with Sass.

Usage

By default Brass will scale to the root font size of your html element, and provide breakpoints based on common device sizes. Use Brass with your own color palette, or import a color library such as Colors.

Optionally, customize the scale and breakpoint settings in brass.scss. Set these options to any custom value and compile using Sass.
