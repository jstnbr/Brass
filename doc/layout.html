Float

Adjust the display of an element by applying classes.

Class Property
.fn float: none
.fl float: left
.fr float: right
.cb clear: both

Width

Use the class .fit to contain an element's max width, or use .fluid to remove max width containment. Use .full to set an element's width to 100%.

Class Property
.fit max-width: 100%
.fluid max-width: none
.full width: 100%

Border Radius

Set the border radius of an element using the class .rad which represents radius. Four total radius options include the default setting and three additional sizes .rad-sm, .rad-md, and .rad-lg.

rad-sm
rad
rad-md
rad-lg
<div class="btn bg-gray rad-sm">rad-sm</div>
<div class="btn bg-gray rad">rad</div>
<div class="btn bg-gray rad-md">rad-md</div>
<div class="btn bg-gray rad-lg">rad-lg</div>
