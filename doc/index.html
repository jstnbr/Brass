A set of simple, discreet classes that includes a grid, type scale, positioning utilities, and base styling for common web elements.

Here's what's included:

  • Typography
  • Grid
  • Layout
  • Button
  • Form
  • Table
  • White Space

Portable

Brass provides a customizable responsive base for new or existing websites in less than 12kb. Supports IE8 and up.

Responsive

Includes a 12-column responsive, mobile first fluid grid system with column nesting, centering, offsetting, and reordering.

Library

Provides control over design elements using class-based styles — includes no colors or effects.

Get Started

Use the minified CSS file, or customize your own build using Sass.
