Grid

Brass grid provides a 12-column responsive, mobile first fluid grid system. Column nesting, centering, offsetting, and reordering are supported.

Columns

Use the grid by opening up a .row and placing column classes inside. Class names are simple — choose the screen size and the number of columns to span. There are four tiers of screen sizes accessable via .xs-c*, .sm-c*, .md-c*, and .lg-c*. Replace * with the number of columns.

Mobile First

By default columns are stacked until defined otherwise. For example, a .xs-c6 column will span half width for all device sizes. A .sm-c6 column will span half width for small devices and up, and stack for extra small devices.

Default responsive breakpoints are provided based on common device sizes, and can be customized in brass.scss.

Spacing

Add .spacing to any row to apply vertical spacing. Each column that is a direct child of that row will receive a bottom margin. 

<div class="row spacing">
    <div class="sm-c6"></div>
    <div class="sm-c3"></div>
    <div class="sm-c3"></div>
    <div class="sm-c4"></div>
    <div class="sm-c4"></div>
    <div class="sm-c4"></div>
    <div class="sm-c2"></div>
    <div class="sm-c2"></div>
    <div class="sm-c8"></div>
</div>

Fluid

Add .fluid to a row to remove its max width containment. Create an entire fluid layout by applying to all outermost rows. 

<div class="row fluid">
  ...
</div>

Gutters

Column gutter width is equal to the root font size of your project. For a tighter fit add .tight to any nested row and column gutters will be reduced by 25%. 

<div class="row tight">
    <div class="sm-c4"></div>
    <div class="sm-c4"></div>
    <div class="sm-c4"></div>
</div>

Nesting

Grids are infinitely nestable. Here is a basic example of how 8 equal columns can be achieved by nesting. 

<div class="row">
    <div class="sm-c6">
        <div class="row">
            <div class="sm-c3"></div>
            <div class="sm-c3"></div>
            <div class="sm-c3"></div>
            <div class="sm-c3"></div>
        </div>
    </div>
    <div class="sm-c6">
        <div class="row">
            <div class="sm-c3"></div>
            <div class="sm-c3"></div>
            <div class="sm-c3"></div>
            <div class="sm-c3"></div>
        </div>
    </div>
</div>

Centering

Center columns with layout utility classes. Use .fn to turn float off and .mx-auto to center. 

<div class="row">
    <div class="xs-c8 fn mx-auto"></div>
</div>

Offsetting

Offset columns at different screen sizes by moving them to the right using .*-offs-* classes. Replace the first * with a screen size tier and the second * with number of columns. 

<div class="row">
    <div class="sm-c4"></div>
    <div class="sm-c4 sm-offs-4"></div>
</div>

Reordering

Change the order of columns at different device sizes using .*-push-* and .*-pull-* classes. Replace the first * with a screen size tier and the second * with number of columns. 

<div class="row">
    <div class="sm-c4 sm-push-5"></div>
    <div class="sm-c3 sm-push-5"></div>
    <div class="sm-c5 sm-push-7"></div>
</div>
