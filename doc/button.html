Button

Brass provides a fundamental button structure for elements that can be customized with utilities for styling.

Basic

The .btn class applies a basic button structure to an element.

Link Button 
<button class="btn">Button</button>
<a class="btn" href="#!">Link Button</a>
<input class="btn" type="button" value="Input Button">

Color

Use your own color palette, or import a color library such as Colors.css.

Button Primary Button Secondary Button 
<a href="#" class="btn bg-silver">Button</a>
<a href="#" class="btn bg-blue white">Primary Button</a>
<a href="#" class="btn bg-green white">Secondary Button</a>

Outline

Create an outlined button by setting the border-color property. This can be done using CSS, or a color library such as Colors.css.

Button Primary Button Secondary Button 
<a href="#" class="btn border--gray gray">Button</a>
<a href="#" class="btn border--blue blue">Primary Button</a>
<a href="#" class="btn border--green green">Secondary Button</a>
