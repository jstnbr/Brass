Table

Brass provides a fundamental structure for tables that can be customized with utilities for styling.

Basic

The .table class creates a basic table structure.

Header Header Header
Data Data Data
Data Data Data 
<table class="table">
  <thead>
    <tr>
      <th>Header</th>
      <th>Header</th>
      <th>Header</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Data</td>
      <td>Data</td>
      <td>Data</td>
    </tr>
    <tr>
      <td>Data</td>
      <td>Data</td>
      <td>Data</td>
    </tr>
  </tbody>
</table>

Border

Add the class .table-b to create a bordered table.

Bordered tables require proper formatting to work correctly, including a thead and tbody.

Header Header Header
Data Data Data
Data Data Data 
<table class="table table-b">
  <thead>
    <tr>
      <th>Header</th>
      <th>Header</th>
      <th>Header</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Data</td>
      <td>Data</td>
      <td>Data</td>
    </tr>
    <tr>
      <td>Data</td>
      <td>Data</td>
      <td>Data</td>
    </tr>
  </tbody>
</table>
