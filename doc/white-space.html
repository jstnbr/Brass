White Space

Brass white space utilities provide a consistent naming convention and scale for positioning elements.

Scale

Use a shorthand naming convention for margin and padding utilities.

Shorthand Value
0 0
h .5rem
1 1rem
1h 1.5rem
2 2rem
3 3rem
4 4rem

Margin

Replace * with a scale shorthand to control the margin of an element in any direction.

Class Style
.m-* Margin
.mx-* Margin X-axis
.my-* Margin Y-axis
.mt-* Margin top
.mr-* Margin right
.mb-* Margin bottom
.ml-* Margin left

Padding

Replace * with a scale shorthand to control the padding of an element. Padding utilities are multi-directional for visual typographic and white space consistency.

Class Style
.p-* Padding
.px-* Padding X-axis
.py-* Padding Y-axis
