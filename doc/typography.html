Headings

The .heading class provides an easy and semantic way to apply additional vertical margin for headings.

List Sizing

Control spacing with .ul-sm, .ul-md and .ul-lg classes, or leave the default spacing. These only affect the lists they are directly applied to, so you can mix and match between nested lists.

  • List item 1
  • List item 2
  • List item 3
    • Nested 1
    • Nested 2
    • Nested 3
  • List item 4 
<ul class="ul ul-lg">
    <li>List item 1</li>
    <li>List item 2</li>
    <li>List item 3</li>
    <ul class="ul ul-sm small">
        <li>Nested 1</li>
        <li>Nested 2</li>
        <li>Nested 3</li>
    </ul>
    <li>List item 4</li>
</ul>

Inline Lists

Set lists inline using the .ul-i class. Use list sizing modifiers for different spacing, or leave the default spacing.

  • List item 1
  • List item 2
  • List item 3 
<ul class="ul-i ul-md">
    <li>List item 1</li>
    <li>List item 2</li>
    <li>List item 3</li>
</ul>

List Style
